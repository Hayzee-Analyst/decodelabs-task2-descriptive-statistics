# Task 2: Basic Statistics using Microsoft Excel

## Project Overview

This project was completed as part of the DecodeLabs Tech Data Analytics Internship.

The objective was to perform descriptive statistical analysis on an e-commerce sales dataset using Microsoft Excel.


## Objective

Calculate the following statistics:

- Mean
- Median
- Count

Summarize important observations from the dataset.


## Dataset Information

The dataset contains customer order information with the following fields:

- Order ID
- Date
- Customer ID
- Product
- Quantity
- Unit Price
- Shipping Address
- Payment Method
- Order Status
- Tracking Number
- Items in Cart
- Coupon Code
- Referral Source
- Total Price

Total Records:
**1,200**


## Tools Used

- Microsoft Excel
- Excel Functions


## Statistical Analysis

### Quantity

- Mean: 2.95
- Median: 3
- Count: 1200

### Unit Price

- Mean: 356.41
- Median: 364.21
- Count: 1200

### Items in Cart

- Mean: 5.49
- Median: 5
- Count: 1200

### Total Price

- Mean: 1053.97
- Median: 823.62
- Count: 1200


## Key Findings

- The dataset contains 1,200 completed customer records.
- Customers purchased an average of 3 items per order.
- Average order value is 1053.97.
- The median order value (823.62) is lower than the mean, indicating the presence of some high-value purchases.
- Online payment was the most frequently used payment method.
- Order statuses were fairly balanced across Delivered, Pending, Returned, Cancelled, and Shipped.
